🔬 인과관계 vs 상관관계 (Confounding Factor Check)

모든 가설 검증 시 4가지 교란변수를 반드시 체크. 보정된 effect = raw - market - season - competition

1️⃣ 시장 전체 상승?

같은 기간 비관리 87K의 N2도 올랐는가? → 로직 변경/시장 전체 상승 착시

2️⃣ 시즌 효과?

같은 기간 해당 업종 전체의 N2가 올랐는가? → 시즌 캘린더 교차 확인

3️⃣ 경쟁사 이탈?

해당 키워드 경쟁사가 이탈/약화했는가? → 상대적 상승 착시

4️⃣ 복합 효과?

다른 투입(리뷰, 블로그)이 동시에 있었는가? → 단일 변수 분리 불가

💡 보정 후에도 양수면 순수 효과 존재. 0 이하면 외부 요인 착시. adjusted_effect 기반으로만 validation.

⚔️ 규칙 충돌 감지 (Rule Conflict Checker)

같은 variable_category의 active 규칙들을 페어로 비교. 방향 반대면 충돌.
